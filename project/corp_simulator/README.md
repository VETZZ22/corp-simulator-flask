# Веб-проект «Симулятор корпорации»

Учебный Flask-проект для практики: интерактивный сайт, где пользователь управляет виртуальной корпорацией, выполняет управленческие действия и наблюдает изменение финансов, персонала, репутации, качества продукта, технологий и рисков.

## Возможности

- главная панель с ключевыми показателями корпорации;
- выполнение управленческих действий через POST-запросы;
- страница стратегии с переименованием корпорации;
- административная панель с журналом событий;
- хранение состояния и истории действий в JSON-файлах;
- готовая структура для публикации в Git.

## Запуск

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

После запуска откройте в браузере:

```text
http://127.0.0.1:5000/
```

## Публикация в Git

```bash
git init

git branch -M main
git remote add origin <ссылка-на-репозиторий>
git push -u origin main
```
